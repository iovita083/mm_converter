# mm_converter

An automated tool to convert mod song packs for *Hatsune Miku: Project DIVA Mega Mix+* to data folders for MicroMix+. Currently not finished although it is working properly. (nvidia gpu is required unless you edit the ffmpeg command in the code)


# How to use

Step one: ensure ffmpeg and python is in your system PATH

Step two: download and setup the latest version of https://github.com/Rikux3/UsmToolkit

Step three: move both the built mm_converter.exe and mm_converter_gui.py to the same folder as UsmToolkit

Step four: run the .py, select the mods and output folders, select the songs to convert, and press "Convert selected", this will take a while if skip video is not checked
